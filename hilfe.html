Datenschutz

Es gibt keinen Server. Importierte Dateien und Einstellungen werden nur in diesem Browser verarbeitet; optional können Sie den Zwischenstand pro Modus (Kursteams, Jahrgangsgruppen, ARGEs) getrennt lokal speichern.

Windows blockiert heruntergeladene .cmd?

Das ist normal (Internetsicherheit / Mark of the Web). Lösung: Rechtsklick auf die Datei → Eigenschaften → unten Zulassen aktivieren → OK. Danach erneut doppelklicken.

Alternativ in PowerShell:

Unblock-File -LiteralPath '…\Downloads\…-Anlage.cmd'
