Schritt 5: PowerShell-Script ausführen

⚠️ Wichtig: Führen Sie das Script im selben Verzeichnis wie die neueteams.csv aus. Admin-Anmeldung erforderlich!

PowerShell-Script:

$TeamsList = Import-Csv -Path .\neueteams.csv -Encoding UTF8
Connect-MicrosoftTeams

$i = 0
foreach ($Team in $TeamsList) {
    $i++
    try {
        $null = New-Team -Template "EDU_Class" -DisplayName $Team.TeamName -MailNickname $Team.Gruppenmail -Owner $Team.Besitzer -ErrorAction Stop
        Write-Host "OK [$i/$($TeamsList.Count)] $($Team.Gruppenmail)"
    }
    catch {
        Write-Warning "Fehler [$i] $($Team.Gruppenmail): $($_.Exception.Message)"
    }
    Start-Sleep -Seconds 2
}

Fertiges Paket zum Doppelklicken

Lädt eine Datei Kursteam-Anlage.cmd (enthält das PowerShell-Skript eingebettet – keine separate .ps1 im Download). Doppelklick startet die Anlage; es erscheint eine Abfrage: interaktive Anmeldung (MFA möglich) oder Benutzername/Passwort. Hinweis: Nach dem Download aus dem Browser kann Windows die Datei zunächst blockieren – dann Rechtsklick → Eigenschaften → Zulassen (Unblock), oder bei SmartScreen „Weitere Informationen“ wählen.

ℹ️ Hinweise:
• Das Erstellen vieler Teams kann mehrere Stunden dauern
• Das Script gibt pro Team eine Zeile aus (OK oder Warnung bei Fehler); Start-Sleep entlastet Microsoft 365
• Fortschritt kann im Microsoft 365 Admin Center verfolgt werden
• Teams haben standardmäßig 5 Jahre (1825 Tage) Ablaufdatum

Klassenliste

ℹ️ Was ist das? Aus dieser Liste werden Microsoft 365-Gruppen (Unified Groups) mit Adressen im Muster <Präfix><Jahr>-<Kürzel>@domain – optional mit Teams und Exchange (Schritt 4). Die Vorschau unten aktualisiert sich beim Tippen (Präfix und Kürzel-Großschreibung links).
Format: pro Zeile Klasse und optional Abschlussjahr, getrennt durch Semikolon, Komma oder Tabulator. Beispiele: 1AK;2030 oder nur 1AK (dann gilt das Standard-Abschlussjahr links). Die E-Mail-Domain der Schule steht oben in der Kopfzeile.
Ergebnis z. B. jg2030-AK@ms365.schule. Graph setzt den Mail-Nickname; die fertige Adresse @… steuern Sie optional über Exchange (Schritt 4).
Gilt für Zeilen nur mit Klassennamen (z. B. 1AK ohne Jahr). Pro Zeile weiterhin möglich: 1AK;2030 – dann überschreibt das Jahr in der Zeile diesen Standard.

Vorschau
