Schritt 5: PowerShell-Script ausführen

⚠️ Wichtig: Führen Sie das Script im selben Verzeichnis wie die neueteams.csv aus. Admin-Anmeldung erforderlich!

PowerShell-Script:

$TeamsList = Import-Csv -Path .\neueteams.csv -Encoding UTF8
Connect-MicrosoftTeams

$i = 0
foreach ($Team in $TeamsList) {
    $i++
    try {
        $null = New-Team -Template "EDU_Class" -DisplayName $Team.TeamName -MailNickname $Team.Gruppenmail -Owner $Team.Besitzer -ErrorAction Stop
        Write-Host "OK [$i/$($TeamsList.Count)] $($Team.Gruppenmail)"
    }
    catch {
        Write-Warning "Fehler [$i] $($Team.Gruppenmail): $($_.Exception.Message)"
    }
    Start-Sleep -Seconds 2
}

Fertiges Paket zum Doppelklicken

Lädt Kursteam-Anlage.ps1 (alle Teams eingebettet + Anmeldeauswahl) und Kursteam-Anlage.cmd. Beide Dateien in denselben Ordner legen, dann die .cmd doppelklicken. Es erscheint eine Abfrage: interaktive Anmeldung (MFA möglich) oder Benutzername/Passwort.

ℹ️ Hinweise:
• Das Erstellen vieler Teams kann mehrere Stunden dauern
• Das Script gibt pro Team eine Zeile aus (OK oder Warnung bei Fehler); Start-Sleep entlastet Microsoft 365
• Fortschritt kann im Microsoft 365 Admin Center verfolgt werden
• Teams haben standardmäßig 5 Jahre (1825 Tage) Ablaufdatum

Manuelles Erstellen (Beispiel):

Connect-MicrosoftTeams
New-Team -Template "EDU_Class" -DisplayName "SS2223 | 3HK | SPWS-SPANR" -MailNickname WS2022-3BK-SPWS-SPANR -Owner mathias.spanring@hak-steyr.at

Jahrgangsgruppen: Grundlagen

ℹ️ Was ist das? Hier werden Microsoft 365-Gruppen (Unified Groups, keine Kursteam-/EDU_Class-Teams) mit E-Mail-Adressen im Muster jg<Abschlussjahr>-<Kürzel>@domain angelegt – analog zu Ihrem Admin-Center.
Ergebnis: jg2030-AK@hak-steyr.at

Klassen und Abschlussjahr

Format: pro Zeile Klasse und Abschlussjahr, getrennt durch Semikolon, Komma oder Tabulator. Beispiel: 1AK;2030 (Klasse 1AK → Abschlussjahr 2030).

Besitzer (Lehrkräfte)

Microsoft Graph: Besitzer müssen als Benutzer im Mandanten existieren (User Principal Name). Pro Klasse eine Besitzer-E-Mail eintragen.
Klasse Gruppen-E-Mail (Vorschau) Mail-Nickname Besitzer (UPN)

PowerShell (Microsoft Graph)

Voraussetzungen: Modul Microsoft.Graph installieren, Connect-MgGraph mit passenden Rechten (z. B. Group.ReadWrite.All, User.Read.All). Es werden Microsoft 365-Gruppen (Unified Groups) erstellt – ohne Teams-Kursteam-Vorlage.
